# n8n AI 자동화 가이드

> 초보자부터 고급자까지, n8n으로 업무 자동화를 마스터하는 실전 가이드

## 책 소개

이 책은 **n8n**을 활용한 AI 자동화의 모든 것을 다룹니다. 코딩 없이도 강력한 자동화 워크플로우를 구축하고, ChatGPT·Claude AI를 연동한 지능형 자동화 시스템을 만드는 방법을 단계별로 안내합니다.

## 대상 독자

- 업무 자동화에 관심 있는 직장인 및 프리랜서
- n8n을 처음 접하는 입문자
- AI와 자동화를 결합하고 싶은 개발자 및 기획자
