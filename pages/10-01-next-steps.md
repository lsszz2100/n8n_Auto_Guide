## n8n 관련 추가 학습 경로

### 보완해야 할 기술

**JavaScript/TypeScript:**
```
- 현재 Code 노드에서 기본적인 JS 사용 가능
- 다음 목표: 비동기 프로그래밍 (async/await)
- 심화: TypeScript로 커스텀 노드 개발
```

**API 이해:**
```
- REST API 설계 원칙 (RESTful)
- OAuth2 인증 흐름 이해
- GraphQL API 연동
- WebSocket 실시간 연결
```

**데이터베이스:**
```
- SQL 쿼리 최적화
- PostgreSQL 고급 기능 (JSON 컬럼, pgvector)
- Redis 캐싱
- 벡터 데이터베이스 (Pinecone, Qdrant)
```

**AI/ML:**
```
- 프롬프트 엔지니어링 심화
- 파인튜닝 기초
- 임베딩과 벡터 검색 이해
- LangChain/LlamaIndex 이해
```

## 커리어 경로

n8n 전문성을 활용한 커리어 방향:

### 자동화 컨설턴트

```
역할: 기업의 업무 자동화 프로젝트 수행
평균 프로젝트 단가: 500만원 ~ 3,000만원
필요 역량:
  - n8n 고급 활용
  - 비즈니스 프로세스 이해
  - 고객 커뮤니케이션
```

### 자동화 개발자 (사내)

```
역할: 회사 내 자동화 팀 소속
업무: 업무 자동화, AI 도구 구축
필요 역량:
  - n8n + 코딩 실력
  - 비즈니스 도메인 지식
  - 운영 경험
```

### SaaS 제품 개발

```
역할: n8n 기반 자동화 제품 창업
예시: 특정 업종(부동산, 법률 등) 특화 자동화 솔루션
필요 역량:
  - n8n 고급 활용
  - 창업 마인드
  - 특정 도메인 전문성
```

## 포트폴리오 프로젝트 아이디어

취업이나 프리랜서를 위한 포트폴리오:

```
1. GitHub에 워크플로우 공개
   - 잘 문서화된 워크플로우 JSON
   - README에 설명과 스크린샷

2. 블로그 작성
   - 자신이 만든 자동화 사례 공유
   - "n8n으로 XX 자동화한 방법" 형식

3. 데모 비디오
   - 워크플로우 실행 화면 녹화
   - 실제 효과(절약된 시간, 처리 건수) 수치화

4. 오픈소스 기여
   - n8n 커뮤니티 노드 패키지 개발
   - npm에 배포
```
