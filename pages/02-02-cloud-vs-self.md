# 레슨 2.2: Cloud vs Self-Hosting 비교

---

## 두 가지 선택지

n8n을 사용하는 방법은 크게 두 가지입니다:

1. **n8n Cloud**: n8n 회사가 운영하는 서버 사용
2. **Self-Hosting**: 내 서버나 컴퓨터에 직접 설치

어느 것이 더 좋다고 단정할 수 없습니다. **상황과 목적에 따라 다릅니다.**

---

## 상세 비교표

| 항목 | n8n Cloud | Self-Hosting |
|------|-----------|-------------|
| **비용** | 월 $20~$50+ | 서버 비용만 (무료 가능) |
| **설정 난이도** | 매우 쉬움 (5분) | 중간~어려움 (30분~2시간) |
| **유지보수** | 자동 (n8n이 관리) | 직접 관리 |
| **업데이트** | 자동 | 수동 |
| **데이터 위치** | n8n 서버 | 내 서버 |
| **실행 횟수** | 요금제에 따라 제한 | 무제한 |
| **커스터마이즈** | 제한됨 | 완전 자유 |
| **보안 통제** | n8n이 관리 | 직접 관리 |
| **가동 시간** | n8n이 보장 | 직접 책임 |
| **스케일링** | 자동 | 직접 설정 |

---

## n8n Cloud 추천 대상

다음에 해당한다면 Cloud를 선택하세요:

✅ **처음 n8n을 시작하는 경우** — 설치 없이 바로 시작 가능
✅ **서버 관리 경험이 없는 경우** — 유지보수 걱정 없음
✅ **빠르게 프로토타입을 만들고 싶은 경우**
✅ **실행 횟수가 많지 않은 경우** (월 5,000회 미만)
✅ **팀이 작거나 개인 프로젝트인 경우**

**요금제 (2026년 기준):**
| 플랜 | 가격 | 실행 횟수 | 워크플로우 수 |
|------|------|-----------|--------------|
| Starter | $20/월 | 2,500회 | 5개 |
| Pro | $50/월 | 10,000회 | 무제한 |
| Enterprise | 문의 | 무제한 | 무제한 |

---

## Self-Hosting 추천 대상

다음에 해당한다면 Self-Hosting을 선택하세요:

✅ **대량의 자동화가 필요한 경우** (월 수만 회 실행)
✅ **개인정보/민감 데이터를 다루는 경우**
✅ **비용을 최소화하고 싶은 경우**
✅ **완전한 커스터마이즈가 필요한 경우**
✅ **기업 보안 정책상 외부 SaaS를 사용할 수 없는 경우**
✅ **개발 및 테스트 환경이 필요한 경우**

---

## Self-Hosting 옵션 3가지

### 옵션 1: 로컬 컴퓨터 (학습용)
- 내 노트북이나 PC에 설치
- 비용: 무료
- 장점: 설치가 간단, 즉시 시작 가능
- 단점: 컴퓨터가 꺼지면 자동화가 멈춤, 외부에서 접근 불가
- 추천: 학습용, 테스트용

### 옵션 2: VPS (Virtual Private Server) (추천)
- AWS EC2, DigitalOcean, Vultr, Hetzner 등
- 비용: 월 $5~20
- 장점: 24/7 운영, 외부 접근 가능, 합리적 비용
- 단점: 서버 관리 지식 필요
- 추천: 개인 프로젝트, 소규모 팀

### 옵션 3: 온프레미스 서버 (기업용)
- 회사 내부 서버에 설치
- 비용: 하드웨어 비용
- 장점: 완전한 데이터 통제
- 단점: 초기 투자 비용, 전문 인력 필요
- 추천: 기업, 보안이 중요한 조직

---

## 추천 VPS 서비스 비교

| 서비스 | 월 비용 | 사양 (n8n 최소) | 특징 |
|--------|---------|----------------|------|
| Hetzner | €4~ | 2GB RAM, 2 CPU | 유럽산, 가성비 최고 |
| DigitalOcean | $6~ | 1GB RAM, 1 CPU | 쉬운 UI |
| Vultr | $6~ | 1GB RAM, 1 CPU | 글로벌 데이터센터 |
| AWS EC2 | $8~ | t3.micro | AWS 생태계 |
| Oracle Cloud | 무료 | 1GB RAM, 1 CPU | 영원히 무료 티어 |

> **n8n 최소 요구사항**: RAM 1GB 이상, CPU 1코어 이상, 20GB 저장공간

---

## 결정 가이드

```
처음 시작이거나 기술이 없다
    → n8n Cloud (Starter 플랜으로 시작)

학습/개발 목적
    → 로컬 설치 (npm 또는 Docker)

실운영, 비용 절감 원함
    → VPS에 Docker 설치

기업, 보안 중요
    → 온프레미스 Self-Hosting
```

## 핵심 요약

- n8n Cloud: 쉽지만 비용 발생, 실행 횟수 제한
- Self-Hosting: 무제한 무료이지만 관리 필요
- 처음에는 Cloud, 규모가 커지면 Self-Hosting 전환 추천
- VPS 중에서는 Hetzner(가성비)나 Oracle Cloud(무료 티어)가 인기

**다음 레슨**: n8n Cloud에서 바로 시작하는 방법을 알아봅니다.
