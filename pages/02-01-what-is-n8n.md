# 레슨 2.1: n8n이란?

---

## n8n 한 줄 정의

> n8n은 앱을 연결하고, 데이터를 이동하며, 업무를 자동화하는 오픈소스 워크플로우 자동화 플랫폼입니다.

2019년 독일에서 Jan Oberhauser가 만든 n8n(읽는 방법: "n-eight-n" 또는 "nodemation")은 현재 전 세계 10만 개 이상의 조직에서 사용되고 있습니다.

---

## n8n의 핵심 철학

n8n은 두 가지 서로 다른 요구를 동시에 만족시킵니다:

"코드 없이도 강력하게, 코드가 있으면 더 강력하게"

| 사용자 유형 | n8n 활용 방식 |
|-------------|--------------|
| 비개발자 | 드래그 앤 드롭으로 400+ 앱 연동 |
| 개발자 | JavaScript 코드 직접 삽입 가능 |
| 팀 | 협업 기능으로 워크플로우 공유 |
| 기업 | 온프레미스 설치로 데이터 완전 통제 |

## n8n vs 다른 자동화 도구 비교

### 주요 경쟁 도구 비교

| 항목 | n8n | Zapier | Make(Integromat) | Power Automate |
|------|-----|--------|-----------------|----------------|
| 가격 | 무료(셀프호스팅) | 유료 | 유료 | 유료 |
| 오픈소스 | O | X | X | X |
| 코드 지원 | JavaScript | X | 제한적 | 제한적 |
| AI 에이전트 | 네이티브 | 부분 지원 | 부분 지원 | 부분 지원 |
| 셀프 호스팅 | O | X | X | 부분 |
| 실행 제한 | 없음(셀프호스팅) | 있음 | 있음 | 있음 |
| 앱 수 | 400+ | 6,000+ | 1,500+ | 1,000+ |
| 커스터마이즈 | 매우 높음 | 낮음 | 중간 | 중간 |

### n8n을 선택해야 하는 경우
- 무제한 실행이 필요할 때
- 데이터를 외부로 보내고 싶지 않을 때 (셀프호스팅)
- 복잡한 로직과 코드가 필요할 때
- 비용을 최소화하고 싶을 때
- AI 에이전트 기능이 필요할 때

### Zapier를 선택해야 하는 경우
- 매우 간단한 자동화만 필요할 때
- 더 많은 앱 커넥터가 필요할 때
- 기술적 설정이 전혀 불가능할 때

## n8n의 주요 기능

### 1. 시각적 워크플로우 편집기
드래그 앤 드롭 방식으로 노드를 연결하여 워크플로우를 시각적으로 구성합니다.

### 2. 400+ 내장 통합
Gmail, Slack, Google Sheets, Notion, Airtable, HubSpot, Stripe, AWS, GitHub 등 400개 이상의 앱과 즉시 연동 가능합니다.

### 3. JavaScript 코드 실행
Code 노드를 통해 JavaScript 코드를 직접 작성하여 어떤 복잡한 로직도 구현할 수 있습니다.

### 4. AI 에이전트 (가장 강력한 기능)
GPT-4, Claude, Gemini 등 AI 모델을 워크플로우에 통합하여 지능형 자동화를 구현합니다.

### 5. 서브 워크플로우
큰 워크플로우를 작은 단위로 분리하여 재사용 가능한 모듈형 자동화를 만들 수 있습니다.

### 6. 실행 기록 및 디버깅
모든 실행 기록을 저장하고, 각 노드에서 어떤 데이터가 흘렀는지 확인할 수 있습니다.

### 7. 팀 협업
워크플로우를 팀원과 공유하고, 역할 기반 접근 제어(RBAC)로 권한을 관리합니다.

## n8n의 버전 이해

### Community Edition (무료)
- 오픈소스, 셀프 호스팅
- 무제한 워크플로우 실행
- 모든 기능 사용 가능
- 직접 서버 관리 필요

### Cloud (유료)
- n8n이 관리하는 호스팅 서비스
- 월 실행 횟수에 따라 요금 책정
- 업데이트, 백업 자동 관리
- 별도 서버 불필요

### Enterprise (기업용)
- 고급 보안 기능
- SSO (Single Sign-On)
- 감사 로그
- 전담 지원

## n8n의 성장 스토리

n8n은 2019년 Jan Oberhauser가 개인 프로젝트로 시작했습니다.

주요 마일스톤:
- 2019: 오픈소스로 첫 공개
- 2020: n8n GmbH 설립, Series A 펀딩
- 2021: 클라우드 버전 출시
- 2022: AI 기능 통합 시작
- 2023: 네이티브 AI 에이전트 기능 대폭 강화
- 2024: 멀티 에이전트, RAG 지원 추가
- 2026: 전 세계 10만+ 조직이 사용하는 주요 자동화 플랫폼

---

## n8n 이름의 의미

n8n = "nodemation"의 약자

- n: node (노드)
- 8: 중간의 8글자 (odematio)
- n: n의 마지막 글자

"node"에서 "n"을 따고, 중간 8자를 숫자 "8"로 줄인 것입니다.
i18n(internationalization), l10n(localization)과 같은 방식입니다.

## 핵심 요약

- n8n은 오픈소스 워크플로우 자동화 플랫폼으로, 코딩 없이도 강력하게 사용 가능
- 셀프 호스팅 시 무료이며 실행 횟수 제한이 없어 비용 효율이 높음
- 400+ 앱 통합, AI 에이전트, JavaScript 코드 실행이 핵심 강점
- Zapier보다 유연하고 강력하지만, 초기 설정이 약간 더 복잡함
