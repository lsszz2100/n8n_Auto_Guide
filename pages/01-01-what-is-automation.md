# 레슨 1.1: 자동화란 무엇인가?

---

## 자동화의 정의

자동화(Automation)란 소프트웨어, 로봇, 기계가 최소한의 인간 개입으로 반복적인 작업을 수행하도록 만드는 것입니다.

일상에서 이미 수많은 자동화를 경험하고 있습니다:
- 은행 앱이 매달 자동으로 이체해 주는 것
- 넷플릭스가 내 취향에 맞는 콘텐츠를 추천해 주는 것
- 이메일 필터가 스팸을 자동으로 분류해 주는 것

비즈니스에서도 마찬가지입니다. 자동화할 수 있는 업무들:

### 영업 부서
- 고객이 문의 폼을 작성하면 → 자동으로 CRM에 등록 + 담당자에게 알림 발송
- 잠재 고객이 이메일을 열면 → 영업팀 Slack에 즉시 알림

### 재무 부서
- 매월 말 → 자동으로 지출 보고서 생성 및 이메일 발송
- 인보이스가 발행되면 → 회계 시스템에 자동 기록

### 마케팅 부서
- 신규 회원 가입 → 웰컴 이메일 시리즈 자동 발송
- 소셜 미디어 예약 포스팅 자동화

### 운영 부서
- 재고가 특정 수량 이하로 떨어지면 → 자동 발주
- 고객 지원 티켓이 들어오면 → 키워드 분석 후 담당팀 자동 배정

## 무엇이 자동화 가능한가?

자동화에 적합한 업무의 특징을 알면, 어디서부터 시작해야 할지 명확해집니다.

### 자동화 가능한 업무의 특징

| 특징 | 설명 | 예시 |
|------|------|------|
| 반복성 | 매번 같은 순서, 같은 작업 | 데이터 복사, 이메일 전송 |
| 단순성 | 창의적 판단이 불필요 | 숫자 계산, 데이터 형식 변환 |
| 규칙성 | 명확한 조건과 결과 | IF 예산 > 1000 THEN 승인 |
| 반복 빈도 | 자주 발생 | 일/주/월 단위 반복 |

### 자동화가 어려운 업무

- 복잡한 인간 감정이 필요한 소통
- 창의적인 아이디어 발상
- 복잡한 윤리적 판단이 필요한 의사결정
- 예측 불가능한 물리적 환경 대응

> 핵심 공식: "X가 발생하면, Y를 실행해서, Z 결과를 얻는다"
> 이 문장으로 표현할 수 있다면, 자동화가 가능합니다.

## 자동화의 3가지 레벨

### 레벨 1: 단순 자동화 (Task Automation)
단일 반복 작업 자동화. 예: 파일 이름 일괄 변경, 데이터 백업

### 레벨 2: 프로세스 자동화 (Process Automation)
여러 단계가 연결된 업무 흐름 자동화. 예: 주문 접수 → 재고 확인 → 배송 알림

### 레벨 3: 지능형 자동화 (Intelligent Automation)
AI와 결합하여 판단까지 자동화. 예: 이메일 내용을 AI가 분석 → 중요도 판단 → 적절한 담당자 배정

n8n은 이 3가지 레벨 모두를 하나의 플랫폼에서 구현할 수 있습니다.

## 자동화가 가져다주는 ROI

자동화는 투자입니다. 만드는 데 시간이 걸리지만, 장기적으로는 엄청난 이득을 줍니다.

### 계산 예시

> 상황: 매일 아침 30분씩 데이터를 엑셀에서 CRM으로 옮기는 작업

| 항목 | 수동 작업 | 자동화 후 |
|------|-----------|-----------|
| 일일 소요 시간 | 30분 | 0분 |
| 월간 소요 시간 | 10시간 | 0시간 |
| 연간 소요 시간 | 120시간 | 0시간 |
| 오류 발생 가능성 | 높음 | 거의 없음 |

> ROI = 절약된 가치 / 자동화 구축 비용
>
> 자동화 구축: 2시간
> 연간 절약: 120시간
> ROI = 6000%

## 핵심 요약

- 자동화란 반복적인 작업을 소프트웨어가 대신 수행하게 만드는 것
- "X가 발생하면 Y를 하여 Z를 얻는다"로 표현 가능한 모든 것은 자동화 후보
- 자동화는 3가지 레벨이 있으며, n8n은 모든 레벨을 지원
- 자동화의 ROI는 매우 높으며, 초기 투자 후 장기적으로 큰 이익을 가져다줌
