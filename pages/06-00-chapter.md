## 6장 학습 방법

각 레슨은 **하나의 완성된 실전 프로젝트**입니다.

직접 따라 만들면서 학습하세요:
1. 목표 확인 → 워크플로우 구조 이해
2. 크리덴셜 설정 → 서비스 연결
3. 단계별 노드 구성 → 전체 워크플로우 완성
4. 테스트 → 활성화

각 프로젝트는 독립적으로 완성할 수 있으며, 원하는 순서로 진행해도 됩니다.
