## 핵심 개념 요약

### AI 에이전트 4가지 유형

| 유형 | 특징 | 사용 예 |
|------|------|---------|
| Reactive | 즉각 반응, 기억 없음 | 빠른 분류 |
| Goal-based | 목표 달성 위해 다단계 행동 | 복잡한 리서치 |
| Utility-based | 최적 선택 | 채널 선택, 우선순위 결정 |
| Learning | 피드백으로 개선 | 고급 지원 봇 |

### AI 통합 핵심

| 노드 | 역할 |
|------|------|
| AI Agent | 도구 + 메모리 + 추론 |
| Basic LLM Chain | 단순 텍스트 생성 |
| Summarization Chain | 긴 텍스트 요약 |
| Q&A Chain | 문서 기반 질의응답 |

### 프롬프트 4요소

1. **명확성**: 정확히 무엇을 원하는지
2. **맥락**: 배경과 상황 정보
3. **제약**: 하지 말아야 할 것
4. **예시**: 원하는 형식의 샘플

### 메모리 유형

| 유형 | 지속성 | 사용 규모 |
|------|--------|-----------|
| Window Buffer | 임시 | 소규모 |
| Postgres | 영구 | 중대규모 |
| Redis | 임시+빠름 | 대규모 |
| Vector Store | 의미 검색 | 장기 기록 |
