## 우리가 함께 배운 것들

### 1장: 자동화 기초
- 자동화가 왜 필요한지, 어떤 업무를 자동화할 수 있는지
- AI 자동화와 규칙 기반 자동화의 차이
- API가 무엇이고 어떻게 시스템들이 연결되는지

### 2장: n8n 시작하기
- n8n Cloud, Docker, npm 세 가지 설치 방법
- n8n 인터페이스 탐색
- 첫 번째 워크플로우 완성

### 3장: 핵심 개념
- 노드, 트리거, 데이터 흐름
- JSON 구조와 표현식 사용법
- HTTP Request로 모든 API 연동

### 4장: 워크플로우 구축
- 조건 분기, 반복 처리, 오류 처리
- 디버깅 전략
- 안정적인 워크플로우 설계 원칙

### 5장: AI 에이전트
- LLM 연동과 AI Agent 노드
- 프롬프트 엔지니어링
- 메모리와 컨텍스트 관리

### 6장: 실전 프로젝트
- Gmail, Slack, Telegram, Notion, Google Sheets 연동
- CRM 자동화, 소셜 미디어 자동화
- 웹훅 기반 실시간 자동화

### 7장: ChatGPT & Claude AI
- OpenAI, Claude API 연동
- AI 이메일 처리, AI 고객 지원 봇
- RAG 시스템, 콘텐츠 생성 자동화
- 멀티 에이전트 시스템

### 8장: 고급 활용
- 서브 워크플로우로 모듈화
- n8n API 활용
- 커스텀 노드 개발
- 성능 최적화 및 보안

### 9장: 운영 관리
- 백업과 복구
- 안전한 버전 업그레이드
- 팀 협업 구조
- 모니터링과 알림
