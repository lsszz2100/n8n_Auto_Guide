## 다음 학습 방향

### 심화 학습 경로

초급에서 고급으로 이어지는 추천 경로입니다:

```
현재 위치 (중급~고급)
    ↓
[심화 1] n8n 커스텀 노드 개발
  - TypeScript로 나만의 노드 만들기
  - npm에 배포하여 커뮤니티에 공유

[심화 2] AI 에이전트 고도화
  - 멀티 에이전트 오케스트레이션
  - 벡터 데이터베이스 연동 (Pinecone, Weaviate)
  - Fine-tuning 모델 활용

[심화 3] 엔터프라이즈 운영
  - Kubernetes 기반 n8n 클러스터 운영
  - 고가용성(HA) 구성
  - 대규모 워크플로우 성능 최적화
```

### 커리어 경로

n8n 실력을 커리어에 활용하는 방법:

```
1. 자동화 컨설턴트
   - 중소기업의 업무 자동화 프로젝트 수주
   - 프리랜서로 시작, 시간당 5~15만원 수준

2. 인하우스 자동화 담당자
   - 회사 내 반복 업무를 자동화하는 역할
   - IT 부서 또는 운영팀에서 수요 증가 중

3. AI 워크플로우 엔지니어
   - AI 도구를 비즈니스 프로세스에 통합하는 역할
   - 최근 가장 빠르게 성장하는 포지션

4. SaaS 제품 개발
   - n8n을 백엔드 자동화 엔진으로 활용한 제품 개발
   - No-code 기반 B2B SaaS
```

## n8n 커뮤니티 참여하기

혼자 공부하는 것보다 커뮤니티에서 배우는 게 훨씬 빠릅니다:

```
공식 커뮤니티:
  community.n8n.io     — 가장 활발한 영어 커뮤니티
  
공식 문서:
  docs.n8n.io          — 최신 기능 레퍼런스

템플릿 라이브러리:
  n8n.io/workflows     — 커뮤니티 공유 워크플로우

YouTube:
  n8n 공식 채널         — 튜토리얼 영상
```

커뮤니티에서 얻을 수 있는 것:
- 내가 겪은 문제를 이미 해결한 사람의 답변
- 다른 사람이 만든 워크플로우 템플릿
- 새로운 버전에서 추가된 기능 정보
- 같은 관심사를 가진 사람들과의 네트워킹

## 마지막 한 마디

자동화의 핵심은 기술이 아니라 **어떤 문제를 해결할지** 아는 것입니다.

n8n을 배웠다는 것은 이제 아이디어를 실제로 구현할 수 있는 능력을 갖췄다는 뜻입니다.

```
좋은 자동화 = 명확한 문제 + 올바른 도구 + 꾸준한 개선
```

처음 만든 워크플로우가 완벽할 필요는 없습니다.
작게 시작하고, 실행하면서 개선하세요.

책을 읽어주셔서 감사합니다.
