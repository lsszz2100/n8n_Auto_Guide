# 8장: 고급 n8n 활용

---

## 이 챕터에서 배울 것

n8n의 기초와 AI 통합을 마스터했다면, 이제 **전문가 수준의 고급 기능**을 다룰 차례입니다.

---

## 챕터 구성

| 레슨 | 제목 | 핵심 내용 |
|------|------|-----------|
| 8.1 | 서브 워크플로우 | 모듈식 자동화 설계 |
| 8.2 | n8n API 활용 | 외부에서 n8n 제어 |
| 8.3 | 커스텀 노드 개발 | 나만의 노드 만들기 |
| 8.4 | 성능 최적화 | 대용량 처리 전략 |
| 8.5 | 보안 강화 | 엔터프라이즈 보안 설정 |

---

## 이 챕터가 필요한 이유

### 규모가 커질수록 구조가 중요합니다

단순 자동화는 하나의 워크플로우로 충분합니다. 하지만 비즈니스가 성장하면:

- **워크플로우 수십 개**: 관리가 복잡해짐
- **팀 협업**: 여러 사람이 동시에 작업
- **재사용성**: 같은 로직을 여러 곳에서 사용
- **성능**: 수천 건의 데이터를 빠르게 처리

이 챕터는 이러한 **프로덕션 레벨**의 요구사항을 해결하는 방법을 다룹니다.

---

## 학습 목표

이 챕터를 마치면:

- **서브 워크플로우**로 재사용 가능한 모듈 설계
- **n8n API**로 외부 시스템에서 워크플로우 실행
- **커스텀 노드**를 직접 개발하여 배포
- **성능 튜닝**으로 대용량 데이터 처리
- **보안 설정**으로 안전한 자동화 환경 구축
