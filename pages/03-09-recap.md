## 핵심 개념 요약

### 노드 유형

| 유형 | 역할 |
|------|------|
| 트리거 노드 | 워크플로우 시작 |
| 일반 노드 | 데이터 처리, 전송 |

### 핵심 트리거

| 트리거 | 사용 시점 |
|--------|-----------|
| Manual | 테스트, 수동 |
| Schedule | 정기 실행 |
| Webhook | 외부 이벤트 수신 |

### 핵심 처리 노드

| 노드 | 역할 |
|------|------|
| Edit Fields | 데이터 생성/변환 |
| IF / Switch | 조건 분기 |
| Merge | 데이터 병합 |
| Split Out | 배열 → 개별 아이템 |
| Filter | 조건 필터링 |
| Code | 자유 JavaScript 코드 |
| HTTP Request | API 호출 |

### JSON 핵심

```javascript
$json.field              // 기본 필드 접근
$json.nested.field       // 중첩 객체
$json.array[0]           // 배열 인덱스
$json.field ?? "기본값"  // null 처리
```

### 표현식 핵심

```javascript
{{ $json.name }}                    // 현재 아이템 데이터
{{ $node["이름"].json.field }}      // 특정 노드 데이터
{{ $now.toFormat('yyyy-MM-dd') }}   // 현재 날짜
{{ $json.amount > 0 ? "양수" : "음수" }}  // 조건 표현
```
